# Sales Dashboard
 
   This Sales Dashboard provides a comprehensive view of a company's sales performance, revenue, and profit metrics. It is designed to help stakeholders quickly analyze trends, identify key profit contributors, and make data-driven decisions. The dashboard includes visualizations for both online and offline sales, regional performance, and item-wise profit and revenue distribution.

# Features

   Key Metrics Overview:
    
   Total Revenue
    
   Total Profit
    
   Profit Margin
    
   Total Orders

   Profit Contribution 

# Revenue and Profit Analysis:

   Total revenue by order year
    
   Total profit by country
    
   Profit by region
    
   Online vs Offline sales comparison

# Product and Sales Insights:

   Unit sales and margin by item type
    
   Total profit and margin by item type
   
   Revenue by order priority

# Interactive Filters:

   Filter data by Country, Region, Item Type, and Sales Channel

# Visual Representations:

   Bar charts, line charts, pie charts, and tables for detailed analysis

# Tools & Technologies

   Power BI for creating interactive dashboards and visualizations
    
   DAX for calculations such as total revenue, profit, and profit contribution
    
   Excel / CSV for source data

# Dashboard Highlights

   Identify top-performing countries and regions
    
   Compare online vs offline sales contribution
    
   Monitor revenue trends by order priority and year
    
   Evaluate product-wise profitability and sales performance
